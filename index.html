Définitions précises

Deux concepts souvent confondus, mais fondamentalement différents.

🌐

L’Internet

L’Internet est un réseau mondial de réseaux informatiques interconnectés qui communiquent grâce au protocole TCP/IP. Il s’agit de l’infrastructure physique et logicielle qui permet à des milliards d’appareils d’échanger des données à travers le monde.

  • Réseau de réseaux
  • Basé sur TCP/IP (1974)
  • Transmet e-mails, vidéos, fichiers, appels, etc.
  • Aucun propriétaire unique
🕸️

Le World Wide Web (Web)

Le Web est un système d’information composé de documents et de ressources hypertextes interconnectés, accessibles via Internet à l’aide d’un navigateur.

  • Basé sur URL, HTTP et HTML
  • Navigation par hyperliens
  • Un service parmi d’autres sur l’Internet
  • Exemples : sites, YouTube, réseaux sociaux

Histoire de l’Internet

1969

Naissance : ARPANET

Premier réseau à commutation de paquets par le DARPA. Premier message envoyé le 29 octobre 1969.

1974

TCP/IP : le langage universel

Vint Cerf et Bob Kahn inventent le protocole qui permet à tous les réseaux de communiquer.

1983

Standardisation officielle

L’ARPANET passe entièrement au TCP/IP le 1er janvier 1983.

1995

Ouverture au grand public

Internet devient accessible à tous. Plus de 50 millions d’utilisateurs.

Histoire du World Wide Web

1989

La proposition de Tim Berners-Lee

Au CERN, Tim Berners-Lee propose un système hypertexte mondial pour partager des documents scientifiques.

1990-1991

Premiers pas

Premier serveur, premier navigateur et premier site web. Le Web est mis à disposition du public le 6 août 1991.

1993 → Aujourd’hui

Explosion mondiale

Navigateur Mosaic (1993). Création du W3C (1994). Naissance du Web 2.0 puis Web 3.0.

La grande distinction

L’Internet est la route. Le Web est une voiture qui circule sur cette route.

🛣️

L’Internet = Infrastructure

Câbles, routeurs, protocoles. Il existe même sans le Web.

🚗

Le Web = Service

Une application parmi d’autres (e-mail, streaming…).

L’Internet existe sans le Web

On peut utiliser l’Internet sans navigateur : e-mail, jeux en ligne, WhatsApp, appels vidéo, BitTorrent…
